Summaries:
This Repository focus on the Medical Navigation.
Under the Research directory, there are the codes to do some research experiments, such as ML algorithms, segementation algorithm, etc.
Under the Tools directory, there are the codes to act as basic tools, such as Dicom file convertor. If you want to test some opensource libraries, you can add your code here.
Other main projects should be put just under the root directory, with Research and Tools.

Warnings:
Please don't add the data(e.g. images, videos) to the Repository since we only have 300M space for free.
Please don't add the third libraries to the Repo. You can add a document to record which library you used.
Please don't add the bin/obj folder.It will be insignificant to track those files.
